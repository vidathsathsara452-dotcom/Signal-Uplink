# Signal-Uplink

SignalUplink is a terminal-based network intelligence and system signal overview tool.
It provides real-time insight into IP routing, interfaces, VPN state, DNS, and system uptime.

## Features

- Public & local IP detection
- Interface status monitoring
- VPN detection (tun / wg / vpn)
- DNS server discovery
- Active connection count
- Clean ANSI UI with banner rendering

git clone https://github.com/Vidath101/SignalUplink.git
